# 1st_mini_proj-radar_sys-
"An Arduino-based Radar System utilizing an HC-SR04 Ultrasonic Sensor and SG90 Servo to map surroundings, featuring real-time visual data processing and distance calculation."
HOW TO RUN :
1.refer the diagram to build the circuit
2.copy paste the code from "radar_sys.ino " to your arduino ide 
3.copy paste the "procesing_radar" to your processing ide
